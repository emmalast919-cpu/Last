Open Source Dependencies
discord.js
The primary library used to interact with the Discord API — commands, events, and gateway.
Apache 2.0
Lavalink
High-performance audio streaming server powering all music playback in Aimo.
MIT
Shoukaku
A stable Lavalink wrapper for Node.js that handles the audio connection layer.
MIT
Node.js
JavaScript runtime environment that powers the entire Aimo backend and bot process.
MIT
OpenAI Node SDK
Used to power the /ai ask command and AI-driven features.
Apache 2.0
Last.fm API
Used to fetch user scrobbles, now-playing, top artists, tracks, and charts for Last.fm integration commands.
API Terms
Mongoose
MongoDB ODM used for playlist storage, user data, and server configuration persistence.
MIT
dotenv
Loads environment variables from .env files for secure token and key management.
ISC
